# Location Feature - User Guide

## 🎯 Quick Start

### For First-Time Users

1. **Login to Dashboard**
   - After logging in, you'll see a beautiful modal asking for location permission

2. **Grant Location Access** (Recommended)
   - Click the "Allow Location Access" button
   - Your browser will show a permission prompt
   - Click "Allow" to enable GPS detection
   - Your location will be automatically detected and displayed

3. **Or Enter Manually**
   - Click "Enter Manually" if you prefer not to share GPS
   - Search for your city in the dropdown
   - Select from the list of popular cities

4. **Skip for Now**
   - Click "Skip for now" to dismiss the modal
   - You can set your location later from the navbar

---

## 📍 Using the Location Selector

### Location Button (Navbar)
- Located in the top navigation bar
- Shows your current location or "Select Location"
- Click to open the location dropdown

### Detecting Your Location
1. Click the location button in navbar
2. Click "Use Current Location" button
3. Grant browser permission if prompted
4. Wait for detection (shows loading spinner)
5. Your location will be displayed automatically

### Searching for a City
1. Click the location button in navbar
2. Type city name in the search box
3. Results filter in real-time
4. Click on a city to select it
5. Location updates immediately

## 🔧 Troubleshooting

### "Location permission denied"
**Solution:** 
- Click the location button
- Use manual search instead
- Or reset browser permissions and try again

### "Location information unavailable"
**Solution:**
- Check if location services are enabled on your device
- Try manual search
- Ensure you have internet connection

### "Location request timed out"
**Solution:**
- Check your internet connection
- Try again after a few seconds
- Use manual search as alternative

### Location not showing in navbar
**Solution:**
- Click the location button
- Select a location manually
- Refresh the page

## 🔐 Browser Permissions

### How to Manage Permissions

**Chrome/Edge:**
1. Click lock icon in address bar
2. Find "Location" permission
3. Change to "Allow" or "Block"
4. Refresh page

**Firefox:**
1. Click lock icon in address bar
2. Click "Connection secure"
3. Find "Location" permission
4. Change setting
5. Refresh page

**Safari:**
1. Safari → Preferences → Websites
2. Find "Location"
3. Change permission for site
4. Refresh page

## ❓ FAQ

**Q: Why do you need my location?**
A: To show you nearby service providers and relevant local results.

**Q: Is my location data safe?**
A: Yes, it's stored only on your device, not on any server.

**Q: Can I use the app without sharing location?**
A: Yes, you can manually enter your city.

**Q: How accurate is GPS detection?**
A: Typically accurate to city/neighborhood level.

**Q: Can I change my location later?**
A: Yes, click the location button anytime to change.

**Q: Does it work offline?**
A: GPS detection requires internet, but saved location works offline.

**Q: Why is detection slow?**
A: GPS can take 5-10 seconds. Ensure good signal and internet.

**Q: Can I save multiple locations?**
A: Currently one location at a time. Multiple locations coming soon!
